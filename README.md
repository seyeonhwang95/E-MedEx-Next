# E-MedEx Next

## MASTER PROMPT v2

This project is a comprehensive monorepo for managing and scaling the E-MedEx Next application.

### Local Development Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/seyeonhwang95/E-MedEx-Next.git
   cd E-MedEx-Next
   ```
2. Install dependencies using pnpm:
   ```bash
   npm install -g pnpm
   pnpm install
   ```
3. Start the services:
   ```bash
   pnpm run dev
   ```
4. Access the application at [demo.localhost](http://demo.localhost)
